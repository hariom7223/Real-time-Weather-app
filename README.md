# Weather App

A simple and interactive weather application that displays real-time weather information for any city using the WeatherAPI service. This project is designed for beginners learning API integration, frontend development, and JavaScript fetch operations.

## Features

- Real-time temperature display  
- Weather condition and description  
- Location-based weather search  
- Responsive and lightweight UI  
- Fetches live data from WeatherAPI

## Project Structure

weather-app/
│
├── index.html
├── style.css
├── script.js
└── README.md

## How to Use

1. Enter the city name and country/state in the input fields.  
2. Click the Search button.  
3. The app will fetch and display:  
   - Temperature  
   - Weather status  
   - City and country details  

## Installation & Setup

### 1. Clone the Repository
git clone <your-repo-link>

### 2. Navigate to the Project Folder
cd weather-app

### 3. Run the Project
Open index.html in your browser.

## API Requirement

This project uses WeatherAPI.com.  
Generate your free API key at:  
https://www.weatherapi.com/

Then, add your key inside script.js:

const apiKey = "YOUR_API_KEY_HERE";
